# フランス旅行準備ガイド

フランス旅行に必要な情報をまとめたウェブサイトです。HTML、CSS、JavaScriptだけで作られており、サーバーなしでブラウザから直接開けます。

## サイトの構成

このサイトは3つのページでできています。

**index.html（トップページ）** は旅行準備の総合ガイドです。パスポートやビザの情報、航空券の取り方、宿泊先の選び方、持ち物リスト、Wi-Fi事情、旅行保険、基本的なフランス語フレーズがひとつのページにまとまっています。

**conversations.html（会話集）** はフランス語の実践的な会話をシチュエーションごとに学べるページです。カフェでの注文、ホテルのチェックイン、買い物、道案内、自己紹介、トラブル対応の6場面が用意されています。それぞれの場面には「基本のやりとり」があり、そこから「おすすめを聞く」「会計を頼む」といった派生パターンに分岐する構成になっています。会話はフランス語、英語、日本語の3言語で表示されます。

**planner.html（旅行プランナー）** は行き先、日数、興味のあるジャンルを選ぶと、自動で旅行プランを組み立ててくれるページです。パリ、リヨン、マルセイユなど複数の地域に対応しており、午前・午後・夜のタイムスロットごとにおすすめスポットが提案されます。

## 使い方

`index.html` をブラウザで開くだけで使えます。各ページ間はナビゲーションリンクで行き来できます。

## 制作の経緯

このサイトはClaude（Anthropic社のAI）との対話を通じて作られました。最初にトップページ、次に会話集、最後に旅行プランナーという順番で、3回のやりとりで完成しています。
